步骤1：准备Ubuntu Linux

LAMP堆栈包括Linux机器……在这种情况下，是Ubuntu。要获得LAMP，您必须首先安装Linux机器……本文假定您已经安装了Ubuntu服务器。 安装Ubuntu服务器后，运行以下命令以更新服务器…

sudo apt update && sudo apt dist-upgrade && sudo apt autoremove

步骤2：安装Apache2 HTTP Server

Apache2 HTTP Server在LAMP堆栈中代表A……这是当今安装的最受欢迎的Web服务器……远远领先于其他受欢迎的Web服务器…… 要在Ubuntu服务器上安装Apache2 HTTP，请运行以下命令…

sudo apt update
sudo apt install apache2

安装Apache2之后，可以使用以下命令停止，启动和启用Apache2服务，使其始终在服务器启动时启动。

sudo systemctl stop apache2.service
sudo systemctl start apache2.service
sudo systemctl enable apache2.service

要测试Apache2设置，请打开浏览器并浏览到服务器主机名或IP地址，您应该看到如下所示的Apache2默认测试页。.看到时，说明Apache2可以正常工作。

http://localhost

步骤3：安装MariaDB数据库服务器

MariaDB在LAMP中代表M，在查看开放源代码数据库服务器时，它是一个不错的起点。 尽管MySQL最初是Linux系统中的默认数据库服务器，但MariaDB已经接管了。 要安装它，请运行以下命令。

sudo apt-get install mariadb-server mariadb-client

安装MariaDB数据库服务器后，以下命令可用于停止，启动和启用MariaDB服务，以便在服务器启动时始终启动。

sudo systemctl stop mysql.service
sudo systemctl start mysql.service
sudo systemctl enable mysql.service

然后，运行以下命令，通过创建root密码并禁止远程root访问来保护MariaDB服务器。

sudo mysql_secure_installation

出现提示时，请按照指南回答以下问题。

Enter current password for root (enter for none): Just press the Enter
Set root password? [Y/n]: Y
New password: Enter password
Re-enter new password: Repeat password
Remove anonymous users? [Y/n]: Y
Disallow root login remotely? [Y/n]: Y
Remove test database and access to it? [Y/n]:  Y
Reload privilege tables now? [Y/n]:  Y

重新启动MariaDB服务器

sudo systemctl restart mysql.service

要测试是否已安装MariaDB，请键入以下命令以登录到MariaDB服务器

sudo mysql -u root -p

然后输入您在上方创建的密码进行登录…如果成功，您应该会看到MariaDB欢迎消息

步骤4：安装PHP和相关模块

LAMP堆栈的最后一个组件是PHP…它是LAMP堆栈中的P…要安装PHP和相关的PHP模块，请运行以下命令…

sudo apt install php libapache2-mod-php php-common php-mbstring php-xmlrpc php-soap php-gd php-xml php-intl php-mysql php-cli php-zip php-curl

安装PHP后，运行以下命令以查找服务器上安装的版本…

php -v

版本号确定PHP默认配置文件的位置…对于PHP 7.2，该位置如下所示：

sudo nano /etc/php/7.2/apache2/php.ini

用安装的 PHP 版本替换上面的版本号。 打开文件后，在文件的以下几行进行更改并保存。下面的值是适用于您的环境的出色设置。

file_uploads = On
allow_url_fopen = On
memory_limit = 256M
upload_max_filesize = 100M
max_execution_time = 360
date.timezone = Asia/Shanghai

完成上述更改后，保存文件并关闭。

步骤4：重新启动Apache2

完成以上对PHP的更改后，请运行以下命令以重新启动Apache2 HTTP服务器以应用PHP设置…

sudo systemctl restart apache2.service

要使用Apache2测试PHP设置，请通过运行以下命令在Apache2根目录中创建phpinfo.php文件

sudo nano /var/www/html/phpinfo.php

然后输入下面的内容并保存文件。

<?php phpinfo( ); ?>

保存文件..然后浏览到服务器主机名，后跟phpinfo.php

http//localhost/phpinfo.php

您应该看到PHP默认测试页…

恭喜你！您已在Ubuntu 16.04上成功安装了LAMP
